# OpenScanStation 0.5.2

## Kodak-Standby in der WebGUI

- Eigener Webbereich für Geräteeinstellungen auf Port 8102.
- Automatische Auflistung erkannter Kodak-i2600-Geräte.
- Anzeige von Verbindung, Backend, Treiberstatus und erkannter Standby-Option.
- Standby aktivieren oder deaktivieren.
- Standby-Zeit zwischen 0 und 240 Minuten konfigurieren.
- Speicherung in `/var/lib/openscanstation/device_settings.json`.
- REST-Endpunkt `/api/device-settings` auf Port 8102.
- Health-Endpunkt `/health` auf Port 8102.
- Eigener gehärteter systemd-Dienst `openscanstation-device-settings.service`.
- Der DEB-Release-Build installiert und aktiviert den Dienst automatisch.

Die Haupt-WebGUI bleibt auf Port 8101. Die Geräteeinstellungen sind unter `http://SERVER-IP:8102` erreichbar.
